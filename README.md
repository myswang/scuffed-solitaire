# Scuffed Solitaire
*a stupid solitaire clone in Lua/Love2D*

## Quick Start

[Install Love2D first.](https://love2d.org/)

```bash
git clone https://github.com/myswang/scuffed-solitaire.git
cd scuffed-solitaire
love .
```
